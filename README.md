Requirements
------------

- cmake (Version >= 2.6.6, http://www.cmake.org)
- Boost (Version >= 1.52.0, http://www.boost.org)
- gtest (optional)
- doxygen (optional)

Generate eclipse project
------------------------

cmake -G "Eclipse CDT4 - Unix Makefiles" <source directory>
